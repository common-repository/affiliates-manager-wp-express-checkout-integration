Process an affiliate commission via Affiliates Manager after a WP Express Checkout payment

== Description ==

This addon allows you to integrate WP Express Checkout with the Affiliates Manager plugin.

When a user pays for an item on your site, this addon will check to see if the user was referred to by an affiliate. It will then give a commission to the affiliate who referred the user.

This addon requires the [Affiliates Manager](https://wordpress.org/plugins/affiliates-manager/) and [WP Express Checkout](https://wordpress.org/plugins/wp-express-checkout/) plugins.

Read the usage instructions [here](https://wpaffiliatemanager.com/affiliates-manager-wp-express-checkout-integration/)

== Installation ==

1. Upload `affiliates-manager-wp-express-checkout-integration` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
